This file contains instructions on how to compile, run, and utilize this
database Management system sotfware.

---------------------------------Compiling and Running-----------------------------

This is a Java Project, run javac DatabaseMS.java to compile the program 
and run java DatabaseMS to run the program 

-----------------------------------Requirements-------------------------------------

*Oracle Database Account Access
*Execute Egarcia.sql on database account. This will create and populate tables, and add
 appropiate trigger and checks.  
*Install appropiate driver if running on local computer. No needed if running on zeus


------------------------------------- Intructions----------------------------------
*Execute Egarcia.sql
*Compile and Run DatabaseMS.java.
*Enter username and password for the database. 
*Follow instruction on the menus to see and manipulate the data.
